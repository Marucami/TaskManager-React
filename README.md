# Task Manager

React-приложение для управления задачами с возможностью:
- Добавления, редактирования и удаления задач
- Системой тегов
- Поиском по задачам
- Сменой темы (светлая/тёмная)
- Дедлайнами с подсветкой просроченных задач
- Сохранением данных между сессиями

## Установка и запуск

1. Установите зависимости:
\`\`\`bash
`npm install`
\`\`\`

2. Запустите приложение:
\`\`\`bash
`npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
\`\`\`
