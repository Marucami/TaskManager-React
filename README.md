# Task Manager

React-приложение для управления задачами с возможностью:
- Добавления, редактирования и удаления задач
- Системой тегов
- Поиском по задачам
- Сменой темы (светлая/тёмная)
- Добавления дедлайнов с подсветкой просроченных задач
- Сохранения данных между сессиями

## Установка и запуск

1. Установите зависимости:

`npm install`


2. Запустите приложение:

`npm start`


Откройте [http://localhost:3000](http://localhost:3000) для просмотра в браузере.
